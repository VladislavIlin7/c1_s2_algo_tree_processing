## Формулировка задачи

Разработать программу на языке C для преобразования математического выражения, заданного в текстовом виде, в двоичное дерево выражения и выполнить преобразование дерева в соответствии с вариантом задания. Для варианта 9 требуется заменить умножение переменной на сумму на сумму произведений. :contentReference[oaicite:0]{index=0}

Преобразование должно выполняться для:
- выражений без преобразуемых элементов;
- выражений с одним преобразуемым элементом;
- выражений с несколькими преобразуемыми элементами, расположенными в разных местах дерева. :contentReference[oaicite:1]{index=1}

## Вариант задания

**Вариант:** 9

**Преобразование:**  
`a * (b + c) -> (a * b) + (a * c)`  
`(b + c) * a -> (b * a) + (c * a)`

## Описание выполненной работы

В программе реализовано построение дерева выражения из строки с использованием двух стеков:

- стек операндов, в котором хранятся указатели на узлы дерева;
- стек операторов, в котором хранятся символы операций и скобок.

Разбор выражения выполняется по алгоритму Дейкстры:
- операнды помещаются в стек операндов;
- операторы помещаются в стек операторов с учётом приоритета;
- при необходимости выполняется свёртка верхнего оператора в поддерево;
- скобки обрабатываются отдельно.

После построения дерева выполняется рекурсивное преобразование по варианту:
- находится поддерево вида `переменная * сумма`;
- строится новое поддерево вида `сумма произведений`;
- аналогично обрабатывается симметричный случай `сумма * переменная`.

Также реализованы:
- динамическое создание узлов дерева;
- копирование поддеревьев;
- вывод дерева в наглядном виде;
- вывод выражения в инфиксной форме;
- освобождение всей выделенной памяти.

## Основные функции программы

- `createNode()` — создание узла дерева;
- `printTree()` — вывод дерева выражения;
- `printExpression()` — вывод выражения в инфиксной форме;
- `freeTree()` — освобождение памяти дерева;
- `copyTree()` — копирование поддерева;
- `getNextToken()` — выделение следующего токена из строки;
- `buildExpressionTree()` — построение дерева выражения;
- `buildTopSubtree()` — свёртка оператора в поддерево;
- `transform()` — выполнение преобразования по варианту;
- `isMultiplyNode()` — проверка, что узел содержит операцию умножения;
- `isPlusNode()` — проверка, что узел содержит операцию сложения;
- `isVariableNode()` — проверка, что узел является переменной.

## Поддерживаемые элементы выражения

Программа обрабатывает:
- переменные;
- целые числа;
- операции `+`, `-`, `*`, `/`;
- круглые скобки.

Преобразование по варианту применяется только к выражениям вида:
- `a * (b + c)`
- `(b + c) * a`

Остальные операции корректно разбираются при построении дерева, но не участвуют в преобразовании.
