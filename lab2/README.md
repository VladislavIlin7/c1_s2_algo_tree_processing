## Формулировка задачи

Разработать программу на языке C для сортировки таблицы и двоичного поиска по ключу.  
Необходимо проверить работу сортировки в трёх случаях:

1. элементы уже упорядочены;
2. элементы расположены в обратном порядке;
3. элементы расположены случайно.

Для каждого случая требуется вывести таблицу до и после сортировки. После сортировки необходимо выполнять двоичный поиск
по ключу.

## Описание выполненной работы

Реализована таблица с составным ключом int и char и строковым значением.  
Ключи и значения хранятся в отдельных динамических массивах.

Основные функции программы:

- `initTable()` — инициализация таблицы;
- `addElement()` — добавление элемента;
- `readTableFromFile()` — чтение таблицы из файла;
- `selectionSort()` — сортировка методом линейного выбора с обменом;
- `binarySearch()` — двоичный поиск по составному ключу;
- `printTable()` — вывод таблицы;
- `freeTable()` — освобождение памяти.

Для проверки используются три файла:

- `sorted.txt` — упорядоченные данные;
- `reversed.txt` — данные в обратном порядке;
- `random.txt` — случайные данные.
